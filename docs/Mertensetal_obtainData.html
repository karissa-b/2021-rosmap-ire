Introduction

This page describes how I obtained the raw RNA-seq data for Mertens et al 2021.

  1. The raw data was desposited in Array Express under the accession number E-MTAB-10352.
  2. I then found the data in ENA
  3. The accession in ENA is Project: PRJEB44542. The total number of files in this project is 114, and includes both RNA- and ATAC-seq data. 4 .I used this project ID as input to the SRA explorer to generate a bash script to download all files.
  4. I then used sed to remove all lines which contain the following ERR accession numbers (i.e. the ones which are ATAC-seq files, which I will not analyse here)
library(tidyverse)
